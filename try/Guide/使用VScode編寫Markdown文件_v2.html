1. 前言

1.1. 什麼是 Markdown？

Markdown 是一種輕量級的標記語言，使用簡單的純文字語法來格式化文件。 它的設計目標是易於閱讀和編寫，並且可以輕鬆地轉換為 HTML、PDF 等格式。

1.2. 為什麼使用 VS Code 編寫 Markdown？

  • 免費且跨平台： VS Code 可以在 Windows、macOS 和 Linux 上使用。
  • 豐富的擴充功能： VS Code 擁有大量的 Markdown 擴充功能，可以增強編輯體驗。
  • 即時預覽： 可以即時預覽 Markdown 文件的效果。
  • 程式碼高亮： 支援各種程式語言的程式碼高亮顯示。
  • 整合 Git： 方便版本控制和協同作業。

2. 安裝 VS Code 和必要的擴充功能

2.1. 安裝 VS Code

  • 前往 VS Code 官方網站：https://code.visualstudio.com/
  • 下載適合你作業系統的版本。
  • 按照安裝指示完成安裝。

2.2. 安裝 Markdown 擴充功能

  1. 開啟 VS Code。
  2. 點擊側邊欄的「擴充功能」圖示 (或按下 Ctrl+Shift+X 或 Cmd+Shift+X)。
  3. 在搜尋框中輸入 "markdown"。
  4. 安裝以下擴充功能：
    • Markdown All in One (Yu Zhang): 提供鍵盤快捷鍵、自動預覽、目錄生成等功能。
    • Markdownlint (David Anson): 提供 Markdown 語法檢查和風格建議。
    • Prettier - Code formatter (Prettier): 用於格式化 Markdown 文件，使其更具可讀性。 (可選，但推薦)
    • Markdown PDF (yzane): 用於將 Markdown 文件匯出為 PDF 格式。

3. 建立和編輯 Markdown 文件

3.1. 建立新的 Markdown 文件

  1. 在 VS Code 中，點擊 "File" -> "New File" (或按下 Ctrl+N 或 Cmd+N)。
  2. 選擇 "Plain Text"。
  3. 點擊右下角的 "Select a language" (或按下 Ctrl+K M 或 Cmd+K M)。
  4. 在搜尋框中輸入 "markdown"，然後選擇 "Markdown"。
  5. 將檔案儲存為 .md 檔案 (例如 my-document.md)。 點擊 "File" -> "Save" (或按下 Ctrl+S 或 Cmd+S)。

3.2. Markdown 基本語法

  • 標題：

    # 一級標題
## 二級標題
### 三級標題

  • 段落：

    這是一個段落。

這是另一個段落。

  • 強調：

    **粗體文字**
*斜體文字*

  • 列表：

    * 項目 1
* 項目 2

1. 第一項
2. 第二項

  • 連結： 用[ ]來寫出現的文字 ( )是 ulr

    [Google](https://www.google.com)
[Youtube](https://www.youtube.com)

    Google
    Youtube

  • 圖片：

    ![Markdown 標誌](https://markdown-here.com/img/icon256.png)
![NASA sunshine on Earth](https://www.nasa.gov/wp-content/uploads/2025/04/54371346378-f762763e96-o.jpg)

  • 影片：

    使用 HTML <iframe> 標籤
步驟：
  找到你想要嵌入的影片的嵌入程式碼。 通常，影片平台會提供一個 "分享" 或 "嵌入" 選項，點擊後會顯示 <iframe> 程式碼。
  將 <iframe> 程式碼複製到你的 Markdown 文件中。
  範例 (YouTube):
    <iframe width="560" height="315" src="https://www.youtube.com/embed/VIDEO_ID" frameborder="0" allowfullscreen></iframe>  
將 VIDEO_ID 替換為 YouTube 影片的 ID。 你可以在 YouTube 影片的 URL 中找到影片 ID (例如：https://www.youtube.com/watch?v=VIDEO_ID)。
優點：
  簡單易用。
  支援大多數影片平台。
缺點：
  需要依賴外部影片平台。
  無法自定義影片播放器的外觀。

  • 程式碼：

    • 行內程式碼： 使用 ` 包圍程式碼。 例如：console.log("Hello")

    • 程式碼區塊： 使用三個反引號 ````` 包圍程式碼，可以指定程式語言。

      function helloWorld() {
  console.log("Hello, world!");
}

  • 表格：

    | 欄位 1 | 欄位 2 |
| ------ | ------ |
| 內容 A | 內容 B |
| 內容 C | 內容 D |

4. Markdown 與程式語言的關係

Markdown 廣泛應用於程式開發領域，用於編寫各種文檔，例如：

  • 程式碼文件： 使用 Markdown 撰寫程式碼的說明文件、API 文檔等。
  • README 檔案： 在程式碼倉庫中，使用 README.md 檔案來介紹專案的功能、使用方法和貢獻方式。
  • 技術部落格： 許多開發者使用 Markdown 來撰寫技術文章和教程。
  • 筆記： 使用 Markdown 記錄學習筆記、會議記錄等。

Markdown 可以與多種程式語言結合使用，例如：

  • Python： 可以使用 Python 腳本來自動生成、修改和轉換 Markdown 文件。
  • JavaScript： 可以使用 JavaScript 庫來在網頁上渲染 Markdown 內容。
  • 其他語言： 其他程式語言 (例如 Java、C++、Go) 也有相應的庫和工具來處理 Markdown 文件。

5. VS Code 的 Markdown 功能

5.1. 即時預覽

  • 按下 Ctrl+Shift+V (或 Cmd+Shift+V) 開啟預覽視窗。
  • 你可以並排顯示編輯器和預覽視窗，以便即時查看效果。

5.2. 自動格式化 (Prettier)

  • 如果安裝了 Prettier 擴充功能，可以自動格式化 Markdown 文件。
  • 按下 Ctrl+Shift+P (或 Cmd+Shift+P) 開啟命令面板。
  • 輸入 "Format Document"，然後按下 Enter。
  • 你也可以設定 VS Code 在每次儲存文件時自動格式化：
    • 開啟 VS Code 設定 (File -> Preferences -> Settings)。
    • 搜尋 "editor.formatOnSave"。
    • 勾選 "Editor: Format On Save"。

5.3. 目錄生成 (Markdown All in One)([TOC])

  • 將游標放在你想要插入目錄的位置 (通常是在文件頂部，標題下方)。
  • 輸入 [TOC] (不含引號)。
  • 儲存文件 (按下 Ctrl+S 或 Cmd+S)。 Markdown All in One 擴充功能會自動生成目錄。
  • 如果沒有自動生成，可以嘗試手動觸發：
    • 按下 Ctrl+Shift+P (或 Cmd+Shift+P) 開啟命令面板。
    • 輸入 "Markdown: Create Table of Contents"，然後按下 Enter。

5.4. 程式碼高亮

VS Code 會自動識別程式碼區塊的語言，並進行語法高亮顯示。 你只需要在程式碼區塊的起始標記中指定程式語言即可：

def hello_world():
  print("Hello, world!")

function helloWorld() {
  console.log("Hello, world!");
}

6. 匯出為 PDF (Markdown PDF)

  1. 安裝 Markdown PDF 擴充功能 (如果尚未安裝)。
  2. 右鍵點擊編輯器中的 Markdown 檔案。
  3. 選擇 "Export (pdf)" 或類似的選項。
  4. Markdown PDF 擴充功能會將 Markdown 文件轉換為 PDF 檔案。

6.1. 設定 Markdown PDF

  • 你可以自定義 PDF 輸出的樣式和格式。
  • 開啟 VS Code 設定 (File -> Preferences -> Settings)。
  • 搜尋 "markdown-pdf"。
  • 你可以修改以下設定：
    • markdown-pdf.displayHeaderFooter: 設定是否顯示頁首和頁尾。
    • markdown-pdf.headerTemplate: 設定頁首的 HTML 模板。
    • markdown-pdf.footerTemplate: 設定頁尾的 HTML 模板。
    • markdown-pdf.format: 設定 PDF 的紙張格式 (例如 "A4", "Letter")。
    • markdown-pdf.orientation: 設定 PDF 的方向 (例如 "portrait", "landscape")。
    • markdown-pdf.style: 設定 PDF 的 CSS 樣式。

6.2. 顯示頁碼

要在 PDF 中顯示頁碼，你需要設定 headerTemplate 和 footerTemplate。 以下是一個範例：

"markdown-pdf.headerTemplate": "<div style=\"width: 100%; text-align: center;\"></div>",
"markdown-pdf.footerTemplate": "<div style=\"width: 100%; text-align: center;\"><span style=\"font-size: 10px;\">Page <span class=\"pageNumber\"></span> of <span class=\"totalPages\"></span></span></div>"

將以上程式碼複製到你的 VS Code 設定中 (settings.json)。

7. 總結

使用 VS Code 編寫 Markdown 文件非常方便，結合 Markdown 擴充功能，可以實現即時預覽、語法檢查、自動格式化、目錄生成和 PDF 匯出等功能。 掌握這些技巧，可以讓你更有效地編寫和格式化文件。

Markdown 在程式開發領域有著廣泛的應用，與各種程式語言都有密切的關係。 學習 Markdown 可以幫助你更好地編寫文檔、管理專案和分享知識。
